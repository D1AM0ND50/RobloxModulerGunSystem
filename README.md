# RobloxModulerGunSystem
A Moduler Gun System for roblox with custom guns support and fully customisable from inside to outside, Anti Cheats applied also.

Hi, I am D1AM0ND5 a Experienced Roblox Animator, Scripter, Visual Effect Desginer and today I brought you my own made Moduler Gun System. A powerfull Future proof system made for FPS games or any game which contains gun as part. This system in made in accordance to real life gun with Perfect Recoils on Verticle and Horizontal Planes and to provide a dynamic View to guns. It have Multi Platform Support for everyone to enjoy it.

Features
- This is a Moduler System means more gun can be added easily without coding.
- 2 Gun are given pre made.
- 3 Different type of fire mode Auto, Semi, burst for best and realistic expereince.
- Multi Platform Support Pc, Mobile, Console.
- Anti Cheatable and clean code structure to expand according to your game.
- Firing + blood Vfx with Blood Toggle for kids.
- Clean Gun UI wirth Ammo UI to see how much Ammo a gun have.
- Fully Customisable Properties for exisiting and custom guns from damage, recoil to Ammo Capacity etc.
- Client Customisation for custom Animation, Sounds for Custom guns
